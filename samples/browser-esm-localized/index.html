Monaco Editor ESM Localization Sample

How to test localization:

  • Right-click in the editor to see the German context menu
  • Press F1 to open the command palette in German
  • Try using Ctrl+F to open the search box with German labels
  • Select different languages from the dropdown and click "Apply Language" to reload with a different localization

Note: This demo shows German localization. To test other languages, you would need to modify the import statement in index.js to import the corresponding language file (e.g., 'monaco-editor/esm/nls.messages.fr.js' for French).
